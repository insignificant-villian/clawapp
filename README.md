## What is clawapp?  

clawapp is a mobile chat app that lets you talk to your AI agent. It works on most smartphones and runs inside your browser. The app connects to the OpenClaw AI Agent platform. You can use it to chat naturally with your AI assistant anytime. The interface supports dark mode and multiple languages, including Chinese.  

You do not need to install complex software or use advanced settings. clawapp aims to make AI chat simple and available on your mobile device.  

## 📱 System Requirements  

- Windows computer with internet access  
- Modern web browser such as Chrome, Edge, or Firefox  
- Smartphone capable of running web apps (Android or iOS preferred)  
- Minimum 2 GB free storage on your smartphone or emulator  
- Stable network connection for smooth chat interactions  

## 🔎 Features  

- Mobile-friendly chat interface that works in browsers  
- Dark mode for comfortable use in low light  
- Supports real-time streaming responses from AI  
- Works via secure WebSocket connection for fast chatting  
- Built on Capacitor framework for cross-platform support

- Multiple languages including Chinese  
- Open source and self-hosted ready

## 💡 How to Use clawapp  

1. Open clawapp either through the installed app, browser, or PWA.  
2. Enter your OpenClaw account details if required.  
3. Start typing your questions or messages in the chat box at the bottom.  
4. The AI will respond instantly and stream replies as you chat.
